Competency Questions back to ToC

This section shows the specification of the Users4RESPOND ontology requirements in the form of Competency Questions:

  • CQ02: Which is the Account of a given Agent?

  • CQ04: Which is the account password of a given Account?
  • CQ05: Which is the Notification Token of a given Account?
  • CQ06: Which is the token value of a given Notification Token?
